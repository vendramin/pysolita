### Description ###
  * pysolita is a very simple solitaire game written in [python](http://www.python.org) and [pygame](http://www.pygame.org/).
  * <a href='http://pysolita.googlecode.com/files/pysolita.jpg'>Here</a> you can see a screenshot.
